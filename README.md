Weather App

A modern weather application that fetches real-time weather data from the OpenWeather API.

This project combines API integration, interactive UI, animations, and dynamic state changes into a responsive weather dashboard experience.

Features

Live weather data from OpenWeather API
Search for any city worldwide
Displays temperature, humidity, wind speed, and conditions
UI Design
Glassmorphism design
3D hover/tilt interaction effect
Smooth transitions and micro-animations
Fully responsive (desktop and mobile)
Dynamic Weather System

UI changes based on real weather conditions:

Clear → calm theme
Rain → particle rain effect
Storm → lightning and storm visuals
Cloudy → muted tones

UX Features

Recent search history
Loading states for smooth feedback
Fast city re-search interaction

Tech Stack

HTML5 — structure
CSS3 — styling, animations, glassmorphism UI
JavaScript (ES6+) — application logic
OpenWeather API — live weather data
Canvas API — particle system animation

API Used

OpenWeather API
https://openweathermap.org/api
